API Documentation

This API returns a JSON file containing user information. The URL you'd use for this is /api/users/[user].

Example Response

{ "user": { "id": 61820378, "name": "Steve0Greatness" }, "stickies": [ { "topic_ID": 604367, "topic_NAME": "Personal Stickies", "topic_OWNER": "Steve0Greatness" } ] }
